Data Cleaning & Visualization Project

Project Objective

The objective of this project is to clean raw data, handle missing values, remove duplicates and outliers, and visualize meaningful insights using Python libraries.

Tools & Libraries Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Google Colab
Dataset

Titanic Dataset

Steps Performed
Imported dataset
Checked missing values
Filled missing values
Removed duplicates
Detected and removed outliers
Performed Exploratory Data Analysis (EDA)
Created visualizations
Generated insights
Saved cleaned dataset
Visualizations
Histogram
Bar Chart
Heatmap
Scatter Plot
Boxplot
Key Insights
Most passengers were between 20–40 years old.
Female passengers had higher survival rates.
Higher fare passengers had better survival chances.
Some columns contained missing values and outliers.
Conclusion

This project helped in understanding data preprocessing, exploratory data analysis, data visualization, and storytelling using Python.
